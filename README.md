# all_shp2gpkg
プロジェクト内のシェープファイルをGeopackageに置き換えるプラグイン

[プラグインのページ](https://plugins.qgis.org/plugins/all_shp2gpkg/)


## プラグインについて
プロジェクトのレイヤのうち、シェープファイルレイヤのみをGeopackageに置き換えします。  
スタイルやレイヤの順序、属性フォームなどはそのまま引き継がれます。  
リレーションは引き継がれませんので再設定が必要です。  

Geopackageファイルは、シェープファイルと同じフォルダに保存されます。  
またレイヤ名は、現在レイヤに使用している名前がそのまま使用されます。
